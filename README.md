Missão Espacial - Calculadora de Viagem Interestelar

Este projeto é um script em Python desenvolvido para calcular a logística de viagens espaciais. Ele permite que o usuário simule o tempo necessário para percorrer distâncias astronômicas com base na velocidade média de uma nave.

🛠️ Funcionalidades

Cálculo de Tempo: Converte a distância (km) e velocidade (km/h) em horas e dias de viagem.

Interatividade: Recebe o nome do astronauta e os dados da missão via terminal.

Precisão: Exibe o tempo de viagem com duas casas decimais para maior exatidão.

💻 Tecnologias

Python 3

Manipulação de tipos de dados (float, input, print)

F-strings para formatação de saída
